Downloading from start

1
Download the game
You'll get a ZIP file
Download Game
2
Open the ZIP file
Find the file you just downloaded and double-click it
3
Drag the "MECCHA CHAMELEON" folder to your Games folder
Important

You must exclude this folder in Windows Defender or the game will not work.

4
Download the maps pack (~6 GB)
Download Maps
5
Open the maps ZIP file
6
Make a folder named "Workshop" in MECCHA CHAMELEON\Chameleon\Binaries\Win64
7
Follow the instructions in the ZIP file

Updating the game

1
Download the game
Download Update
2
Drag contents into your existing MECCHA CHAMELEON folder
Replace files when prompted

Starting the game

1
Make sure Steam is running
Open Steam before launching the game
2
Open PenguinHotel.exe
Found in MECCHA CHAMELEON\Chameleon\Binaries\Win64
